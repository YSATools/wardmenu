facecards
===

  0. Install NodeJS <http://nodejs.org>

  1. Install facecards

        git clone git://github.com/coolaj86/facecards.git
        cd facecards
        npm install
        node deploy
        node bin/server 9000

  2. Open <http://localhost:9000>
